## 报错问题

启动HSF应用时，hsf.log日志文件报错信息如下。


```inform7
[Weighting Component] Grouping rule xxx parsed failed
```

```prolog
[Weighting Component] Invalid grouping rules received.
```

## 报错原因

权重规则解析失败、应用规则失败等错误。

## 解决方案

检查权重规则的配置项。如果在`hsf-config.log`日志文件中，可以查看到`[Weighting Component] Weighting rule for service:[${uniqueName}] parsed OK: ${weightingRule}`，表示配置成功。
