## 说明

目前OpenSearch对底纹进行的数据统计指标各包括：核心指标，流量指标，点击指标， 引导搜索指标，四个维度：

**核心指标**

| **指标名** | **定义** | **计算公式** | **依赖** | **意义** |
| --- | --- | --- | --- | --- |

|     |     |     |     |     |
| --- | --- | --- | --- | --- |
| **指标名** | **定义** | **计算公式** | **依赖** | **意义** |
| 底纹PV | 请求底纹且返回成功的次数，即不包括被限流等原因导致返回错误的请求 | 每请求且返回成功一次记1 |  | 当日请求底纹（且返回成功）的次数 |
| 底纹UV | 请求底纹且返回成功的用户数，即不包括被限流等原因导致返回错误的请求 | 同一个用户无论请求多少次都只记1 | 依赖通过SDK/API请求时，底纹请求中上传了user\_id参数 | 当日请求过底纹的用户数 |
| 底纹请求报错次数 | 由于被限流等原因导致返回错误的请求次数 | 每请求报错一次记1 |  | 反映底纹是否需要扩容或进行其他调整 |
| 底纹PV-CTR | 每次底纹请求的平均点击率 | 底纹IPV/底纹PV | 依赖通过SDK/API应用搜索时，搜索请求设置raw\_query参数和 from\_request\_id参数 | 底纹的点击情况 |
| 底纹UV-CTR | 底纹的点击用户率 | 底纹IPV\_UV/底纹UV | 依赖通过SDK/API请求时，搜索请求设置raw\_query参数和 from\_request\_id参数，以及底纹请求上传user\_id参数 | 用户对底纹结果的点击情况 |
| 引导搜索PV占比 | 点击底纹进行搜索的次数占比 | 引导搜索PV/搜索PV | 依赖通过SDK/API应用搜索时，搜索请求中设置了from\_request\_id参数 | 底纹引导的搜索用户数 |
| 引导搜索PV-CTR | 底纹引导搜索的点击率 | 底纹引导搜索的搜索IPV/引导搜索PV | 依赖通过SDK/API应用搜索时，搜索请求设置raw\_query参数和 from\_request\_id参数依赖开通数据采集功能，并且上传了用户点击行为数据 | 衡量底纹引导搜索的召回、排序效果 |

## 使用介绍

- 单日数据默认展现最近一天各个数据指标的情况。可通过日期控件选择展示指标的日期。目前数据采集数据产出暂时为T+1天，上传行为数据后第二天才能查看报表。

- 数据对比以折线图的方式默认展现最近7天各个数据指标的变化。可通过日期控件选择展示指标的时间跨度。可通过点击图表中指标的icon勾选在图表中展示的指标项。

- 查看方法：点击导航栏“统计报表—->底纹报表”，选择应用、模型名称、指标类型及时间，查看统计图表。
