AI试衣-图片精修模型 (aitryon-refiner) 采用按自然月（即每月从1号到月底）累计用量的阶梯定价。您的月度使用量越高，超出部分的单价越优惠。

- 前25张：25 \* 0.30 = 7.5元

- 第126到200张（共75张）：75 \* 0.25 = 18.75元
