阿里云百炼提供丰富的应用模板，您可以免费测试应用模板的效果，并通过模板一键创建应用。应用支持多种发布渠道，并且可以通过 API 调用。

## 步骤一：选择并测试模板

阿里云百炼提供丰富的应用模板。所有模板均支持 **免费** 在线测试， **不消耗您的额度。**

1. 访问 [阿里云百炼控制台](https://bailian.console.aliyun.com/?tab=app#/app-market/newTemplate)，单击需要测试的应用模板，此处以 **公众号爆文工厂** 为例。

![20251014204114](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2109540671/p1016000.jpg)

2. 可以通过预设指令或自定义指令测试应用模板效果，输入“人工智能改变世界”，模板生成了对应的公众号文章和配图。

![20251014210825](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2109540671/p1016003.jpg)


## **步骤二：创建应用**

测试完成后，可复制模板，创建可独立配置和管理的应用。

1. 点击模板页面右侧 **复制此应用**，填写 **基础信息**，点击 **配置清单** 中未开通服务右侧的 **开通**。

![20251014211244](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2109540671/p1016004.jpg)

2. 在服务页面，点击右上角 **立即开通**，根据提示填写信息，点击 **开通服务**。

![20251014211602](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2109540671/p1016005.jpg)

3. 开通对应服务后，在模板页面，点击 **刷新**，完成配置清单后单击 **复制**，完成应用创建。


> 复制模板不产生费用，但后续在该应用中进行问答或API调用将消耗额度。


![20251014211809](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2109540671/p1016006.jpg)

4. 应用创建后保存在 [应用管理](https://bailian.console.aliyun.com/#/app-center) 页面，可进行调用和编辑。

![20251014212118](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/2109540671/p1016007.jpg)


## **步骤三：发布应用**

应用复制后默认为 **已发布** 状态，能够通过 [API调用](https://help.aliyun.com/zh/model-studio/dashscope-api-reference/ "")，单击 **发布渠道** 可查看所有可用选项，各渠道的使用说明请参见 [应用分享](https://help.aliyun.com/zh/model-studio/share-an-application "")。
