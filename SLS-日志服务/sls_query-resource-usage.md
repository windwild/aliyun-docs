操作步骤

阿里云支持导出日志服务的使用明细。本文介绍查询日志服务使用明细的操作步骤。

## 操作步骤

1. 登录[费用与成本](https://billing-cost.console.aliyun.com/)。

2. 在左侧导航栏，选择**账单** \> **账单详情**。

3. 在 **用量明细** 页签中，完成如下设置。




| **参数** | **说明** |
| --- | --- |



|     |     |
| --- | --- |
| **参数** | **说明** |
| **产品名称** | 选择 **sls**。 |
| **计量规格** | 选择 **日志服务**。 |
| **使用时间** | 选择您要查询的时间范围。<br>仅支持自然月内的用量查询。例如开始时间为2022-02-01，结束时间最大为2022-02-28，不支持选择2022-03-01。 |
| **计量粒度** | 选择 **小时**。 |

4. 单击 **导出。**


待文件导出成功后，您可以将文件下载到本地，查看日志服务的使用明细。
