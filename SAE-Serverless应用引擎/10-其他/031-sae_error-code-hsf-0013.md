## 报错信息

配置HSF的route单元化参数后，报错信息如下。


```routeros
invalid route while WriteMode=unit, route : ${route}, route parameter should be: long
```

## 解决方案

检查配置的route单元化参数所对应的类型，该参数只能是long类型或java.lang.Long对象类型。
