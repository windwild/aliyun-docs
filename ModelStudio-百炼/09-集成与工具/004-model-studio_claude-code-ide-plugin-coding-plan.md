阿里云百炼 Coding Plan 支持 Claude Code IDE 插件，可参考本文进行配置与使用。

## **安装 Claude Code 并配置 Coding Plan**

Claude Code IDE 插件依赖于 Claude Code CLI 工具，请先参考 [Claude Code](https://help.aliyun.com/zh/model-studio/claude-code-coding-plan "") 完成 Claude Code 的安装与 Coding Plan 的配置。

## **安装使用 Claude Code IDE 插件**

Claude Code IDE 插件支持在 VSCode、VSCode 系列 IDE（如 Cursor、Trae 等）、JetBrains 系列 IDE（如 IntelliJ IDEA、PyCharm 等）中使用。

Claude Code for VS Code插件

Claude Code for Jetbrains插件

1. 打开VS Code，在扩展市场中搜索 `Claude Code for VS Code` 并安装。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9898540771/p1053283.png)

2. 安装完成后，重启 VSCode。单击右上角图标进入 Claude Code。

![截屏2026-02-06 17](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9898540771/p1053286.png)

3. 在对话框中输入`/`，选择 General config 进入设置页面，在Selected Model中设置 [支持的模型](https://help.aliyun.com/zh/model-studio/coding-plan-overview "")，新建一个新窗口开始对话。

![截屏2026-02-06 17](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9898540771/p1053288.png)


1. 打开JetBrains（如 IntelliJ IDEA、PyCharm 等），在扩展市场中搜索 `Claude Code` 并安装。
